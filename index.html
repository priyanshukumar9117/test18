SQL: Creating Database and Table

1. Create a Database named "library":
CREATE DATABASE library;
2. Delete the "library" Database:
DROP DATABASE library;
3. Create a Table "book_records":
CREATE TABLE book_records ( Book_Library_No INT PRIMARY KEY, Book_Name VARCHAR(100), Author_Name VARCHAR(100), Book_Edition VARCHAR(50), Price DECIMAL(10, 2) );
4. Delete the "book_records" Table:
DROP TABLE book_records;
